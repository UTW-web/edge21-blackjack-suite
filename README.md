# EDGE 21: BlackJack
EDGE 21 is a Blackjack analytics platform featuring EV calculation, gameplay simulation, and strategy analysis. It assumes perfect basic strategy and supports advanced scenarios including card counting.

The BlackJack module originated as a personal project while learning card counting and has evolved into a broader analytical toolkit. Updates are in development.

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.0.5.
